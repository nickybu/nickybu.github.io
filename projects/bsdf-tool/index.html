A Tool for Constructing and Visualising Bidirectional Scattering Distribution Functions

University
 Mon, May 11, 2020 Dissertation Java Rendering
Full-text Review Framework Viewer Renderer

Submitted in partial fulfillment of the requirements for the degree of B.Sc. Information Technology (Software Development) (Hons.) at The University of Malta (2018).

Keywords: Physically-based rendering, Bidirectional Reflectance Distribution Functions, Monte Carlo integration

Languages/Frameworks/Tools: Java, LWJGL, Sunflow renderer

Abstract

The current state-of-the-art with regards to constructing and visualising Bidirectional Scattering Distribution Functions (BSDFs) for physically-based rendering (PBR) is rather constrained, especially when wishing to employ the arbitrary light scattering models in a particular renderer. In this project, a framework to construct arbitrary Bidirectional Reflectance Distribution Functions (BRDFs) is designed, complemented by a visualisation tool. A renderer’s functionality is extended to support the BRDF Framework, producing any renders required for the evaluation.

Image quality metrics and a verifier built into the framework are utilised to demonstrate sound physically-plausible implementations of the BRDFs, having no difference in image quality when compared to images produced using the renderer’s standard re!ectance models.
